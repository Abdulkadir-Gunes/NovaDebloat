📌 Core Features & Capabilities
NovaDebloat offers comprehensive tools that touch every corner of your system. Here are the solutions we provide by category:

🗑️ 1. UWP (Bloatware) Apps Cleanup
Get rid of the pre-installed apps that come with Windows and unnecessarily consume your RAM and CPU in the background.

Safely uninstalls system apps like Cortana, Maps, Weather, Solitaire Collection, and Xbox.

Improves your system's boot speed and saves valuable disk space.

🤖 2. AI & Copilot Disabling
Prevent unwanted AI assistants from draining your system resources.

Completely disables the Windows Recall feature and the Copilot sidebar.

Turns off the integrated AI tools in Paint and Notepad, allowing you to return to the classic experience.

Frees up your hardware resources by stopping background AI services.

🛡️ 3. Privacy & Telemetry Protection
Your personal data belongs to you. Stop Windows from sending data to Microsoft servers in the background.

Blocks the Customer Experience Improvement Program (CEIP) and automatic data transmission.

Disables system error reporting, Ad ID tracking, and cloud searches in the start menu.

🎮 4. Gaming Performance Optimizations
Minimize FPS drops, stuttering, and input lag in your games.

Xbox DVR: Stops background game recording to lighten the system load.

Mouse Acceleration: Disables mouse acceleration, which is critical for competitive gaming, ensuring pinpoint accuracy.

Hardware-Accelerated GPU Scheduling (HAGS): Boosts performance on supported graphics cards.

Network Latency Reduction: Optimizes the Nagle algorithm to improve ping (ms) times in online games.

🚀 5. Performance Boost & UI Improvements
Free your system from heavy visual loads and speed up your workflow.

Disables visual animations and transparency effects to provide an instantly responsive Windows experience.

Replaces the sluggish Windows 11 context menu with the highly preferred Classic Right-Click menu.

Adds a "Take Ownership" option to the right-click menu, allowing you to instantly gain administrative rights over files.

🛠️ 6. System Repair & Maintenance Tools
Check your system's health and clean up digital junk with a single click.

Cleans up accumulated Temp (temporary) files and old Event Logs on your disks.

Automatically repairs corrupted Windows system files by launching SFC and DISM scans directly from the interface.

🚀 Installation & Running (How to Use)
NovaDebloat runs on a powerful PowerShell backend accompanied by a modern HTML5/CSS3/JS frontend. You have two options to launch the application:

Method A: With the Pre-Compiled EXE (Recommended & Easiest Way)
Find the NovaDebloat.exe file in the downloaded folder.

Right-click and select "Run as Administrator".

The tool will start its local server in the background and automatically open the modern optimization panel in your default web browser (in Microsoft Edge app mode).

Method B: With the PowerShell Script (For Advanced Users)
Right-click the Start-Debloater.ps1 file in the folder and select "Run with PowerShell".

Alternatively, open a PowerShell console as an administrator, navigate to the script's directory, and run it directly.

Once the server is active, the interface will load in your browser.

⚠️ CRITICAL NOTE: The ui folder containing the interface files must always be in the same directory (folder) as the NovaDebloat.exe or Start-Debloater.ps1 files! Otherwise, the server will not be able to locate the interface files, and the program will fail to run properly.

🛑 Security & Usage Warnings
First Step - Restore Point: When you open the application for the first time, it is highly recommended to click the blue "Create Restore Point" (Geri Yükleme Noktası Oluştur) button located at the top right of the Dashboard. This way, if you don't like the changes made, you can revert your system to its previous state in seconds.

Risk Levels: Exercise caution when applying settings labeled as 'Danger' (Kritik Risk) or 'Caution' (Yüksek Risk) in the interface. For instance, completely turning off Windows Updates or disabling Windows Defender can leave your system vulnerable. Only use these settings if you know exactly what you are doing and have alternative security measures in place.
