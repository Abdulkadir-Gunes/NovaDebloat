Features
Modern local dashboard opened in an app-style browser window
English and Turkish language support
Safe, Gaming, Privacy, Full Debloat, and Fresh Install presets
UWP app removal with optional Winget fallback
Privacy and telemetry reduction
AI/Copilot/Recall related controls
Windows service management
Gaming latency and Game DVR/Game Bar tweaks
Performance and visual effect optimization
Network, Windows Update, DNS, and Delivery Optimization controls
Explorer, desktop, taskbar, notification, and sound tweaks
Dry-run preview before applying selected tweaks
Per-tweak risk, impact, action, status, and restart-required badges
Category-local search and risk filtering
Restore Defaults page for common reversible settings
Re-apply previous profile
Export/import selected tweak profiles as JSON
Before/after system snapshot
HTML report export
Usage
Download Nova Debloat 2.0.exe.
Right-click and choose Run as administrator.
Create a restore point from the dashboard.
Choose a preset or select tweaks manually.
Use Preview to review what will be applied.
Click Apply Selected Tweaks.
Review the Live Console and exported report.
Restart Windows if the applied tweaks require it.
Safety Notes
Safe and Moderate tweaks are generally lower risk.
Caution tweaks may remove apps or disable services.
Danger tweaks can affect Windows Update, Defender, Firewall, Edge, Store, networking, audio, or core security behavior.
Some Windows-protected components may not be removable even when a command runs.
App removals may require Microsoft Store or Winget to reinstall.
Service and policy changes may require a reboot.
Presets
Preset	Purpose
Safe	Low-risk privacy, UI, and quality-of-life changes
Gaming	Game DVR/Game Bar, latency, power, and performance-focused tweaks
Privacy	Telemetry, ads, AI integrations, and tracking reduction
Full Debloat	Broad debloat selection while avoiding the most dangerous options
Fresh Install	Recommended cleanup for a newly installed Windows system
Profile Export / Import
Nova Debloat 2.0 can export your selected tweaks as a JSON profile. You can import the same profile on another machine or use Re-apply Last after Windows updates.
